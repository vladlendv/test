New command

  • git init (create git in directory)
  • echo >> README.mn (create readme file)
  • git config --global user.name "Name" (create username and email. --global or --local / user.name "" or user.email q@q)
  • git add -A (add file in index, -A = all)
  • git commit -a -m "description" (commit changes. -a = all, -m = message)
  • git status (check repository status)
  • git log (see all changes history)
  • git log --pretty=format:"%h %ad | %s%d [%an]" --graph --date=short (change history in comfortable format)
  • command --help (open documentation for this command)
  • next we need create new repository on GitHub and to follow instruction
  • git checkout master (change current directory)
  • git push repo-name master (push changes on GitHub)
